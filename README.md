# What is the MotorPH Payroll System?
Based on Java with Maven, this payroll system, once running in your terminal, allows you to perform the following:
1. If you are an Employee, view your details after providing your ID
2. If you are a Payroll Staff, perform payroll calculations for a chosen cutoff period from June to December 2024
   either by bulk (all employees) or by single employee (by providing ID)
Note: The program will display an error if your two CSV files (in resources folder) are not in the same directory as MotorPH.Java
DO NOT change the names of the files as this will cause a fatal error in the code.
